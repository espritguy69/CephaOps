# Service Installer App (PWA)

Mobile-first PWA for on-site SIs:

- Job list
- Navigation / map link
- GPS capture
- Status updates (OnTheWay, MetCustomer, Completed)
- Camera photo capture
- Serial scanning
- Material usage logging
- Offline mode (IndexedDB)

# Cephas Service Installer Operations System (CSIOS)

## Features
- Automated Email Parsing
- Order Management
- Scheduler
- Inventory & Serialized Material Tracking
- Splitter Port Management
- KPI & Status Engine
- Dockets & Invoicing
- Reporting Dashboards
- Full Settings Module (CRUD)

## Tech
- ASP.NET Core 8 (API)
- React (Frontend)
- SQL Server (DB)
- Azure/AWS (Storage)
